# VGA Controller for HD Screen

## Описание

Этот проект реализует VGA контроллер для вывода изображения на HD экран (например, 1280x720) с использованием FPGA. Он генерирует сигналы горизонтальной и вертикальной синхронизации (HSYNC и VSYNC), а также управляет цветами RGB. Проект предназначен для работы с разрешением 1280x720 пикселей и выводит красный прямоугольник на экране с чёрным фоном.

Проект использует PLL (Phase-Locked Loop) для генерации тактовой частоты, необходимой для правильной синхронизации VGA.

## Функции

- **Генерация VGA сигналов:** Горизонтальная и вертикальная синхронизация (HSYNC и VSYNC).
- **Цветовое пространство:** Красный компонент RGB.
- **Поддержка HD разрешения:** 1280x720 пикселей (HD).
- **Использование PLL для генерации тактового сигнала.**
- **Прямоугольник в центре экрана, заполненный красным цветом.**

## Структура проекта

- `vga_red.v`: Основной модуль VGA для отображения изображения на экране.
- `vga_pll.v`: Модуль PLL для генерации тактовой частоты.

## Схема подключения

Проект предназначен для FPGA, поддерживающего вывод на VGA экран. Требуется подключить сигналы:
- **HSYNC** и **VSYNC** к соответствующим выводам VGA.
- **RGB** (красный компонент) к входу экрана.

## Требования

- FPGA (например, Altera Cyclone IV или аналогичные).
- VGA монитор с поддержкой разрешения 1280x720.
