# OpenEP4CE6-C FPGA Development Board

This repository contains resources, guides, and example code for working with the **OpenEP4CE6-C** FPGA development board from Waveshare, featuring the **Altera Cyclone IV EP4CE6** chip. The board is designed for a variety of I/O interfaces and FPGA-based development projects.

## Features

- **FPGA Chip**: Altera Cyclone IV EP4CE6
- **Onboard Interfaces**:
  - I2C, SPI, UART
  - PS/2, VGA, LCD, SD Card
  - USB, Ethernet, and more
- **Pinout and Schematics**: [View Pinout](https://www.waveshare.com/wiki/OpenEP4CE6-C)

## Quick Start

To get started with your OpenEP4CE6-C board, follow the user manual to power up and configure the device. 

## Project Demos

Here are some example projects you can run with the OpenEP4CE6-C FPGA board:

1. **Light Up LEDs**: Simple LED blink demo to verify basic board operation.
2. **Joystick Demo**: Use the onboard joystick to control actions.
3. **8 Push Buttons Demo**: Interact with 8 push buttons for input.
4. **8 Segment LED Display Demo**: Display numbers or characters on the LED display.
5. **4x4 Keypad Demo**: Interface with a 4x4 keypad.
6. **DS18B20 Temperature Sensor Demo**: Read and display temperature data.
7. **Buzzer Demo**: Sound the buzzer with FPGA logic.
8. **PS/2 Keyboard Demo**: Use a PS/2 keyboard for text input.
9. **VGA Monitor Demo**: Display graphical output on a VGA monitor.
10. **LCD1602 & LCD12864 Demos**: Display text and graphics on LCD screens.
11. **USB Communication Demo**: Setup communication with USB devices.
12. **SD-Card Demo**: Read and write data to an SD card.
13. **Ethernet Control Demo**: Send and receive data via Ethernet.
14. **I2C EEPROM Demo**: Interface with an I2C EEPROM for data storage.

For more details on each demo, refer to the user manual and accompanying code in the repository.
